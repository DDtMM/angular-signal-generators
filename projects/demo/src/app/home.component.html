Angular Signal Generators

Angular Signal Generators are purpose built signals meant to simplify common tasks encountered in Components. They work like vanilla signals, but they have extra capabilities built in to help extend a signal's functionality. Most can be either used in a similar fashion to a writable signal, or be passed a reference to an existing signal to generate new values.

You can install these from npm npm install @ddtmm/angular-signal-generators

Demos

  • Debounce Signal

    Creates a signal whose value isn't changed until it a certain has passed since the last update.
    Example
    Debounced {{debounced()}}

  • Extend Signal

    Adds new methods to an existing signal.
    Example
    const calculator = extendSignal(5, {
  multiply: (s, multi) => s.update(x => x * multi);
});
calculator.multiply(3);
console.log(calculator()); // 15;

  • Lift Signal

    "Lifts" methods from a signal's value to the signal itself just by passing a tuple of method names.
    Example
    const liftedArray = liftSignal([1, 2, 3], ['push']);
liftedArray.push(4)
console.log(liftedArray()); // [1, 2, 3, 4];

  • Map Signal

    Creates a signal whose input value is immediately mapped to a different value based on a selector. Either a value or multiple signals can be passed and used in the selector function.
    Example
    Doubled {{mapDoubled()}}

    The Sequence Signal is useful for situations where you want to easily cycle between options.
